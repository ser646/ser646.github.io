TF2 Trade Helper

What is TF2 Trade Helper?

TF2 Trade Helper is a browser extension which add new elements to steam trade, profile and popup window. This extension has been created just by me - ser646. I've created it to help myself trade faster and be more efficient. Now i want to share it with you! :D

Features:

  • Metal calculator. (+ , - , x , /) - popup window
  • Inventory status (keys, ref, rec, scrap) - using apikey - popup window
  • Add multiple type of item at once
  • Show current amount of metal in inventory
  • Dedicated metal and keys adding buttons
  • Accept (LMB) or Accept & Send (RMB) - trade button
  • Backpack.tf link button on steam profile
  • and more to come!

If you have any idea or you've spotted a bug - post it on steam group or add me on steam! :D
